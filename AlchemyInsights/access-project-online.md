# <a name="access-project-online"></a>Access-Project Online

- Jos Project Online, tilaa ensin Project Online [](https://docs.microsoft.com/ProjectOnline/get-started-with-project-online) lisäämällä se Office 365 ja varmista sitten, että [sinulle on](https://docs.microsoft.com/ProjectOnline/step-1-sign-up-for-project-online#next-make-sure-you-can-get-in)määritetty käyttöoikeus .
- Näiden [ohjeiden avulla](https://docs.microsoft.com/ProjectOnline/step-2-add-people-to-project-online) voit lisätä käyttäjiä Project Onlineen ja jakaa [Project Online](https://docs.microsoft.com/ProjectOnline/step-2-add-people-to-project-online#4-finally-share-project-online-with-the-people-you-added) heidän kanssaan.

Voit antaa käyttäjille Project Web App -sivuston käyttöoikeustilan SharePoint avulla:

1. Kirjaudu Project Online järjestelmänvalvojana.
2. Valitse Project Web App -aloitussivulla  Jaa oikeassa yläkulmassa.
3. Kirjoita **Jaa Project -valintaikkunassa** kutsu henkilöitä -ruutuun sen henkilön käyttäjänimi, jonka käyttöoikeudet **haluat antaa.**
4. Valitse **Näytä asetukset** ja valitse sitten Valitse **käyttöoikeustaso -valikosta** Project Web **Appin Project esimiehet.**
5. Valitse **Jaa**.
