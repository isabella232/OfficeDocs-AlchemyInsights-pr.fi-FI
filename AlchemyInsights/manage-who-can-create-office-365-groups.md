# <a name="manage-who-can-create-office-365-groups"></a>Office 365 -ryhmien luontioikeuksien hallinta

Oletusarvon mukaan kaikkien organisaatioon kuuluvien käyttäjien luoda ryhmiä O365.
  
Voit halutessasi muuttaa PowerShellin, joka rajoittaa käyttäjien alijoukkoa ryhmän luomisen avulla voit luoda suojausryhmän.
  
Noudata vaiheittaisia ohjeita: [kuka voi luoda Office 365-ryhmien hallinta](https://support.office.com/article/4c46c8cb-17d0-44b5-9776-005fced8e618)
