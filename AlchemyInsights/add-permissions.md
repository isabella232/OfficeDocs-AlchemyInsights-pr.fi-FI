# <a name="add-permissions"></a>Käyttöoikeuksien lisääminen

1. Järjestelmänvalvojan suostumuksen päätepisteen avulla voit määrittää sovelluksen staattisesti [pyydettyjen käyttöoikeuksien luettelon.](https://docs.microsoft.com/azure/active-directory/develop/v2-permissions-and-consent#to-configure-the-list-of-statically-requested-permissions-for-an-application)
1. Sovellusten rekisteröintien luomisoikeuksien myöntämiseen on käytettävissä kaksi käyttöoikeutta, joista jokaisella on eri toimintatapa. Lisätietoja on kohdassa Sovelluksen [rekisteröintioikeudet mukautetuille rooleille Azure Active Directory.](https://docs.microsoft.com/azure/active-directory/roles/custom-available-permissions)
1. [Roolin käyttöoikeudet:](https://docs.microsoft.com/azure/active-directory/roles/permissions-reference#role-permissions)taulukot, jotka kuvaavat kunkin roolin Azure Active Directory käyttöoikeuksia. Joillakin rooleilla voi olla lisäkäyttöoikeuksia Microsoft-palvelut ulkopuolella Azure Active Directory.
