# <a name="sharepoint-online-throttling"></a>SharePoint Onlinen rajoittaminen

SharePoint Online käyttää rajoittamista SharePoint Online -palvelun optimaalisen suorituskyvyn ja luotettavuuden ylläpitämiseen. Rajoittaminen rajoittaa käyttäjien toimien tai samanaikaisten kutsujen määrää (komentosarjojen tai koodin avulla), jotta resurssien liikakäyttö estetään. Jos ajasta käytetään vain 99 prosentin ajasta, se johtuu mukautetusta koodista.
  
Lisätietoja estosta on kohdassa Estäminen tai estäminen [SharePoint Onlinessa.](https://go.microsoft.com/fwlink/?linkid=2022019)
