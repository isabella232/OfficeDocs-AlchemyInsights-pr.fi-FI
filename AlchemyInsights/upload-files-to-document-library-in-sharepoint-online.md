# <a name="upload-files-to-onedrive-or-sharepoint"></a>Upload tiedostojen OneDrive tai SharePoint

Voit ladata tiedostoja tiedostokirjastoon SharePoint Onlinessa vetämällä ja pudottamalla ne tietokoneesta tiedostokirjastoon. Kun OneDrive, voit synkronoida tiedostoja tietokoneen ja pilvipalvelun välillä, jotta voit käyttää tiedostojasi missä tahansa. Voit myös käyttää synkronoidut tiedostot suoraan Resurssienhallinnassa ja käyttää tiedostoja myös offline-tilassa. Lisätietoja lataamiseen ja synkronointiin on alla olevien linkkien kautta.
