# <a name="customize-a-session-host-image-for-windows-virtual-desktop"></a>Istunnon isäntäkuvan mukauttaminen Windows virtuaalityöpöydällä

Virtuaalikoneen (VM) valmisteleminen virtuaalisen kiintolevyn perusnäköistiedoston avulla on kaksi tapaa virtuaalityöpöytää Windows varten:

1. [Luo virtuaalikone hallitun kuvan avulla Azuressa](https://go.microsoft.com/fwlink/?linkid=2127906)ja siirry sitten ohjelmiston [valmisteluun ja asennukseen.](https://go.microsoft.com/fwlink/?linkid=2128064)
1. [Luo kuva paikallisesti lataamalla](https://go.microsoft.com/fwlink/?linkid=2128065) kuva, [valmistellaan Hyper-V VM](https://go.microsoft.com/fwlink/?linkid=2127907)ja mukauta se tarpeidesi mukaan.

Lisätietoja on ohjeaiheessa [VHD-peruskuvan valmisteleminen ja mukauttaminen.](https://go.microsoft.com/fwlink/?linkid=2127838)
