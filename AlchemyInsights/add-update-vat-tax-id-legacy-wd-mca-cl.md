# <a name="add-or-update-vattax-id---legacy-wd--mca-cl---recommended-steps"></a>ALV-/verotunnuksen lisääminen tai päivittäminen – vanha WD + MCA CL – suositellut vaiheet

Verotunnusta käytetään verovapauslaskelmien laskemiseen, ja se näkyy laskussasi.

1. Siirry Kustannusten [hallinta + laskutus -sivulle.](https://ms.portal.azure.com/#blade/Microsoft_Azure_GTM/ModernBillingMenuBlade/Overview) 
2. Valitse **Ominaisuudet** vasemmalta. 
3. Valitse **Verotunnus-ruudussa** **Verotunnusten** hallinta ja kirjoita sitten uusi verotunnus.
4. Valitse **Päivitä**. 

Jos et näe Verotunnukset-ruutua, verotunnuksia ei vielä kerätä alueellesi, tai portaalin verotunnuksia ei tueta tilissäsi.
