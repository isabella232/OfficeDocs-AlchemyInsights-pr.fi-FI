# <a name="enable-seamless-sso"></a>Saumattoman SSO:n ottaminen käyttöön

Azure Active Directory (Azure AD) Seamless Single Sign-On (Seamless SSO) kirjautuu automaattisesti sisään käyttäjiin, kun he ovat yrityksen työpöytäversioissa, jotka on yhdistetty yrityksen verkkoon.
  
[Ota Seamless SSO käyttöön noudattamalla tässä artikkelissa annettuja ohjeita](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
