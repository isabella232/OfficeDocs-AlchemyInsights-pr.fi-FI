# <a name="error-message-when-sharing-files-from-windows-explorer"></a>Virheilmoitus, kun tiedostoja jaetaan Windows Resurssienhallinnasta

Kun yrität jakaa Windows Resurssienhallinnasta, näyttöön voi tulla seuraava virhesanoma:
  
"Valitettavasti palvelimeen ei saada yhteyttä juuri nyt. Yritä myöhemmin uudelleen"
  
Voit ratkaista virhesanoman valitsemalla Internet  Explorerissa Työkalut Internet-asetukset-kohdasta Lisäasetukset-välilehden ja tyhjentämällä sitten Älä tallenna salattuja sivuja \>  \> **levylle -valintaruudun.** 
  
 **Huomautus:** Sinun on ehkä käynnistettävä selain uudelleen, jotta asetus tulee voimaan. Lisätietoja on kohdassa ["Palvelimeen ei saada yhteyttä juuri nyt"](https://go.microsoft.com/fwlink/?linkid=2022914)-virhesanoma tiedostojen jakamisen jälkeen.
